gopress
----

This is a fast blog framework,powered by [Go](https://golang.org)

It is currently under development.

## Features

- Fast generating
- Easy to use

## Installation

```bash
go get github.com/qichengzx/gopress
```

## Quick Start

```bash
gopress
```
